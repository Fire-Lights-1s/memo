# 파일 내용 읽기
```sh
more 파일명
less 파일명
```
- **h** : 명령어상태에서 사용할수 있는 키 도움말 확인하기
- **Enter** : 한 행식 아래로 이동하기
- **Space Bar** : 한 화면씩 아래로 이동하기(f와 동일)
- **f** : 한 페이지씩 아래로 이동하기
- **b** : 한 페이지씩 위로 이동하기
- **=** : 현재 위치의 행번호 보기
- **/문자열** : 지정한 문자열을 검색하기
- **n** : /문자열로 검색한 문자열을 차례대로 계속해서 찾기
- **!쉘명령어** : 명령어상태에서 쉘명령어를 실행하기
- **v** : 명령어로 열려있는 파일의 현재위치에서 vi실행하기
- **q** : 명령어 종료하기
