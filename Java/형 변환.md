**자료형 변환(Type Cast)** : 변수 내에 저장된 데이터의 자료형을 변경시킨다.
1. 자동(묵시적) 형 변환 : 작은 크기의 자료형을 큰 크기의 자료형으로 저장할 경우
2. 강제(명시적) 형 변환 : 변경할 자료형을 정확하게 지정해서 변환하는 경
3. 접미사 형식 : 리터럴 상수일 경우 접미사 형태로 자료형을 변환
```
// 자동(묵시적) 형 변환 
byte a = 127;
int b = a;

// 강제(명시적) 형 변환
double c = 127.0;
byte d = (byte)c;

// 접미사 형식
float e = 10.5f;
long f = 100l;
```
