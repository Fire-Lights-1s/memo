#### 항과 연산자
1. 단항 연산자 : 항이 한 개인 연산자(ex. num++)
2. 이항 연산자 : 항이 두 개인 연산자(ex. num1 + num2;)
3. 삼항 연산자 : 항이 세 개인 연산자(ex. (3>5) ? 1 : 0;)
#### 대입 연산자
- 변수에 값을 대입하는 연산자
```java
	int age = 24;
	int sum = 12 + 12;
```
#### 산술 연산자

| 연산자 | 기능                       |
| --- | ------------------------ |
| +   | 두 항을 더함                  |
| -   | 앞의 항에서 뒤의 항을 뺌           |
| *   | 두 항을 곱함                  |
| /   | 앞의 항에서 뒤의 항을 나누어 몫을 구함   |
| %   | 앞의 항에서 뒤의 항을 나누어 나머지를 구함 |
