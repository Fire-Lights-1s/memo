## 이미지 빌드
IaC(Infrastructure as Code)
- 인프라 상태를 코드로 관리
- 프로그램이 코드를 읽어 인프라 관리

Dockerfile
- 이미지를 만드는 단계를 기재한 명세서
- 원하는 이미지 상태를 코드로 작성
- docker build 명령을 통해 이미지 생성
