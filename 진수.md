### 10진수 <-> 2진수 
123 = 1 x 100 + 2 x 10 + 3 x 1
      = 1 x 10^2 + 2 x 10^1 + 3 x 10^0

(2진수)111 = 1 x 2^2 + 1 x 2^1 + 1 x 2^0
               = 1 x 4 + 1 x 2 + 1 x 1
               = 4 + 2 + 1
               = 7
